# 쿵야 머지 계산기
 넷마블 휴대폰 게임 쿵야 머지 아일랜드의 조합 계산기입니다. 동작되는 실제 웹 사이트는 <http://merge.banipest.com/>에서 확인 가능합니다.
 
## 기술스택
 * [react-create-app](https://create-react-app.dev/ )
 * [mui](https://mui.com/)
